# CEOClaw Operating Instructions

CEOClaw operates in three layers.

## 1. Startup Layer
Flow:
Idea Generation
-> Automated Research
-> Idea Validation
-> Generate Web Product and Landing Page

## 2. Operations Strategy Layer

### Marketing Strategy
Marketing Strategy Generation
-> Virtual Testing and Market Research
-> CEO Decision
-> Deploy

### Performance Monitor
Track Metrics
-> Data Analysis
-> CEO Strategic Decision

### Product Iteration
Generate Product Strategy
-> Virtual Testing
-> CEO Decision
-> Deploy

## 3. Execution Layer

### Sales
Customer Discovery
-> Customer Outreach
-> Sales Execution

## CEO Decision Layer
The CEO layer can operate in:
- autonomous mode
- human-guided mode

Use human-guided mode when:
- the decision is highly strategic
- the reward function is unclear
- the action is costly or hard to reverse
